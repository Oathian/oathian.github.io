My Projects

API Project

This was my first project, a restful API built with JavaScript using node, express, PostgresSQL and tested using Jest.

The major challenge I faced with this project was with version control as this was the first time I had used the github branches feature. With practice however, I overcame these difficulties and have dealt with merge conflicts better in my later projects.

I am pleased with how I handled my SQL database queries and table joins, before the project I would have thought these would have caused the most issues. I am also pleased with the way I wrote the test to check if a review is deleted.

Discussion Board Project

This project was a discussion board built in React using HTML, CSS and React hooks.

There were a few challenges along the way most of them CSS based. I found it difficult to get the image and colour gradient of the background to work as I wanted though I got the desired result in the end.

I think the spinning loading sign fits well with the theme of the site and I think the design of the sorting menu looks neat considering how messy it could have been.

Sailing App Project

An app that tracks sailing experience built in TypeScript in React native with expo. Testing was done in Cypress and Jest, AWS cognito was used for authentication

This was a group project built in 8 days with 3 other aspiring junior devs. Using agile practices: morning stand ups with a daily scrum leader, features were broken down into tickets for the Kanban board. Wireframes and component diagrams were drawn up in Figma. A large part of the project was using technologies that were new to all of us so there was a phase of spiking out these new technologies.
